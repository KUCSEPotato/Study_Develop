# Study_Develop

개발 학습 및 알고리즘 문제 풀이를 위한 저장소입니다.

## 프로젝트 구조

### 1. Algorithm
알고리즘 및 자료구조 학습 내용을 챕터별로 정리한 디렉토리입니다.

- **Chapter 1**: 이진 탐색, 외판원 문제
- **Chapter 2**: 선택 정렬
- **Chapter 3**: 재귀 알고리즘
- **Chapter 4**: 퀵 정렬
- **Chapter 5**: 해시 테이블 (커스텀 구현, SHA256)
- **Chapter 6**: 그래프 탐색 (BFS - 망고 판매상 찾기)
- **Chapter 7**: 허프만 코딩, DFS 트리 순회, 파일 디렉토리 탐색
- **Chapter 8**: 트리 자료구조 (AVL 트리, 이진 트리, B 트리, Splay 트리)
- **Chapter 9**: 다익스트라 알고리즘
- **Chapter 10**: 탐욕 알고리즘
- **Chapter 11**: 동적 프로그래밍
- **Chapter 12**: KNN 알고리즘

### 3. BE_Study
백엔드 프레임워크 학습 디렉토리입니다.

#### Flask_Study
Flask 웹 프레임워크 학습 자료

**Chapter 2 - Flask 기초**
- 기본 애플리케이션 구조
- Blueprint를 통한 모듈화
- Cookie & Session 관리
- Logging 설정
- Request & Response 처리
- Routing (HTTP 메서드, URL 변수, URL 빌더, 타입 힌트)
- 정적 파일 제공
- 템플릿 시스템 (상속 포함)

**Chapter 3 - Flask 고급**
- 인증 시스템 (Session, Flask-Login)
- 데이터베이스 CRUD 작업
- Flask-Migrate를 통한 DB 마이그레이션
- RESTful API 구현

#### Study_FastAPI
FastAPI 프레임워크 학습 자료

**Chapter 3**
- 기본 FastAPI 애플리케이션
- 테스트 코드

## 기술 스택

- **Language**: Python
- **Backend Frameworks**: Flask, FastAPI
- **주요 학습 주제**: 알고리즘, 자료구조, 웹 개발, RESTful API

## 학습 목표

1. 알고리즘 및 자료구조에 대한 깊은 이해
2. 백준 문제 풀이를 통한 코딩 실력 향상
3. Flask와 FastAPI를 활용한 웹 애플리케이션 개발 역량 강화
4. 클린 코드 작성 및 베스트 프랙티스 습득

## 시작하기

```bash
# 저장소 클론
git clone https://github.com/KUCSEPotato/Study_Develop.git

# 가상환경 생성 (권장)
python -m venv venv
source venv/bin/activate  # macOS/Linux
# 또는
venv\Scripts\activate  # Windows

# 필요한 패키지 설치
pip install flask fastapi uvicorn
```

## 참고사항

- 각 챕터 및 문제별로 독립적인 Python 파일로 구성되어 있습니다.
- 알고리즘 학습은 주로 "Hello Coding 알고리즘" 책을 참고하여 작성되었습니다.
- Flask 학습은 단계별로 기본부터 고급 기능까지 다룹니다.
